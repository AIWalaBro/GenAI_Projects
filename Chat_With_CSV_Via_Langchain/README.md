## 2. Setup of Project

```bash
    - git clone 'your repository link'
    - conda create -p venv_genai python=3.10 -y
    - conda activate venv_genai
    - cd CHAT_WITH_CSV_WITH_LANGCHAIN

```

## Installation

```bash
    - pip install -r requirements.txt
    - streamlit run main.py
```
